# highView Extended Hardware Abstraction Layer Library

Part of the highView Library Suite developed for Pervasive Displays [EPD Extension Kit Gen 3 (EXT3 or EXT3-1)](https://www.pervasivedisplays.com/product/epd-extension-kit-gen-3-EXT3/) and iTC screens [monochrome](https://www.pervasivedisplays.com/products/?_sft_product_colour=black-white) or colour [black-white-red](https://www.pervasivedisplays.com/products/?_sft_product_colour=black-white-red) and [black-white-red-yellow](https://www.pervasivedisplays.com/products/?_sft_product_colour=black-white-red-yellow)

This library provides a light hardware abstraction layer with C-like syntax for C and C++ SDKs.

## Features

+ GPIO
+ Wire I2C
+ SPI
+ Serial
