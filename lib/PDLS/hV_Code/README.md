# Bar- and QR-code library

Part of the Pervasive Displays Library Suite for Pervasive Displays [EPD Extension Kit Gen 3 (EXT3)](https://www.pervasivedisplays.com/product/epd-extension-kit-gen-3-EXT3/) and iTC [monochrome](https://www.pervasivedisplays.com/products/?_sft_etc_itc=itc&_sft_product_colour=black-white) and [colour](https://www.pervasivedisplays.com/products/?_sft_etc_itc=itc&_sft_product_colour=black-white-red) screens.

This library provides bar-code 39 and 128, and QR-code for the Pervasive Displays screens.

The extended version support an I2C GPIO expander.

!!! warning
 The QR-code library requires the external QRCode library
 See
 * https://www.arduino.cc/reference/en/libraries/qrcode/
 * https://github.com/ricmoo/QRCode

## Features

+ Bar-code 39
+ Bar-code 128-B
+ QR-code
